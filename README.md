## Prerequisites

[Pyenv](https://github.com/pyenv/pyenv)  
[Intro to Pyenv](https://realpython.com/intro-to-pyenv/)
[Pipenv](https://realpython.com/pipenv-guide/)

## Installation

```sh
git clone <repo>
cd <repo>
export PIPENV_VENV_IN_PROJECT=1   
pipenv install --three
```
